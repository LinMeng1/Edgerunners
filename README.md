# Edgerunners
包含

## NightCity
模块化离散PC管理平台终端  
  
基本功能  
- 用简洁的界面占据尽量少的桌面空间，减少对使用者的影响  
- 提供弱连接的远程控制解决方案  
- 对PC的硬件信息进行上传及监控  
- 即时消息传递  
- 模块管理  

开发者支持  
- 增加了适用于VS上的项目模板  
- 提供供模块测试的Modhub项目
